# Game Overview
Subject 04 is a retro low-poly first-person horror game where players explore dangerous environments inhabited by creatures that are invisible to the naked eye. To reveal their true positions, players must rely on various visual devices that expose what cannot normally be seen. By using these tools to perceive a hidden reality, players must avoid unseen threats and find a way to survive.
